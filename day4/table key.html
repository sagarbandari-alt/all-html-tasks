EMMETS

Emmets:

*emmets are shortcut abbreiations for writing the html and css code

*Emmet Abbreviation saves you time and effort while allowing you to focus on more important matters.

Emmets table

Shortcut

(commands)

Description
"!" To create a skeleton of HTML, type exclamation mark(!) and click the 'Tab' key
">" To create child elements using '>'
"+" To create Sibling elements using '+'
"." To create implicit tag name>
"{}" To create text within an element using {} character
"*" To create the code multiple times using *
"$" To add numbers using $
" " To show the custom attributes
div>(p>img)+p Grouping is a clear way to group elements to avoid an unexpected error
"^" add element to the parent
